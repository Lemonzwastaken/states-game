# U.S. States Guessing Game

A simple Python game where you guess all 50 U.S. states. Correct answers are displayed on the map.

## PANDAS

* This is a more simpler project compared to what ive worked on the past 2 projects because ive been working on this using pandas
also I wanted to experiment more so yeah, enjoy :3

## Requirements

* Python 3
* pandas

Install pandas:

```
pip install pandas
```

## Run the Project

```
python main.py
```

Or run the compiled version:

```
dist/main.exe or github release
```

## How to Play

* Enter a state name in the input box
* Correct guesses appear on the map
* Exit by pressing Cancel or submitting empty input

## Output

After the game ends:

```
evaluation/analysis.csv
```

This file shows which states were guessed or missed.
